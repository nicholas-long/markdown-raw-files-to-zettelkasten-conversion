# x64 linux C style calling conventions

`RDI, RSI, RDX, RCX, R8, R9`
- Arguments 1-6 are passed via registers `RDI, RSI, RDX, RCX, R8, R9` respectively;
- Arguments 7 and above are pushed on to the stack.
