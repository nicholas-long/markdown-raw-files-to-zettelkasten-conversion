# vim hotkeys and tricks

useful vim hotkeys and tricks

- get a zettel id from a file path `df/;D`
- `gea` add to previous word
- markdown headings - `0v$r-`
    - 0 optional
- `d/` delete until...
  three backticks = delete snippet
- `d]` `{` `}` moving paragraphs quickly

```
gep
```
